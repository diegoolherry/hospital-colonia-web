# Hospital Colonia ASSE - Dr. Samuel Bertón

Landing page institucional desarrollada para el Hospital Colonia ASSE, ubicado en Colonia del Sacramento, Uruguay.

## Descripción

Sitio web informativo con secciones de servicios, instalaciones, contacto y emergencias. Desarrollado de forma independiente por iniciativa propia.

## Tecnologías utilizadas

- Next.js
- React
- JavaScript
- HTML & CSS

## Demo en vivo

https://hospital-colonia.vercel.app/

npm run dev
